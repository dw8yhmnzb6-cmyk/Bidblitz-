## Architecture
Frontend: React + Tailwind + Leaflet | Backend: FastAPI | DB: MongoDB | Server: IONOS 212.227.20.190

## Match-3 Game (March 2026)

### Features
- 8x8 Grid with 6 gem types (colored symbols)
- Click-to-swap mechanics with match validation
- Auto-cascade: matches clear → gems drop → new matches check
- Combo multiplier (x1, x1.5, x2...)
- 8 progressive levels (target score 500 → 15000)
- 60s timer per level + move counter
- Highscore in localStorage
- Game Over / Level Complete overlays
- No login required (public game)
